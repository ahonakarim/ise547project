CSV Analyst Assistant

A course project that turns plain-English questions about CSV data into a structured query, validates it against the dataset, and runs deterministic Pandas code. The language model is used for intent parsing only; execution stays in audited backend code.

View source on GitHub README & setup

Interactive app

This site is static (GitHub Pages). The live UI runs on Streamlit Cloud. Embedding Streamlit in an iframe often fails (redirect loops via share.streamlit.io, cookies, or security headers), so we open the app in a new tab instead.

Launch Streamlit app (new tab)

What it does

The Streamlit app loads the same logical datasets as benchmarks and scripts: data/processed/<id>_cleaned.csv if present, else data/raw/<id>.csv, else optional large-file paths from .env (see README).

  • Summary statistics (mean, max, count, …)
  • Grouped aggregations (e.g. by region, category)
  • Filtered aggregations (conditions on columns)
  • Time series at day, week, or month granularity

How it works

1. User question → LLM produces JSON matching a fixed schema 2. Validator checks columns, types, and task rules 3. Pandas runs the approved operation; charts use Matplotlib in the UI layer
