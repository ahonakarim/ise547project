CSV Analyst Assistant

Ask questions about tabular data in plain English. The app maps each question to a structured query, validates it against your dataset, and runs deterministic Pandas code (no arbitrary model-generated execution).

Run the interactive app

The Streamlit UI needs a Python host. Free options:

  • Streamlit Community Cloud (connect this repo, entrypoint app/main.py, add secrets for API keys if needed)
  • Hugging Face Spaces (Streamlit SDK)
  • Your laptop: pip install -r requirements.txt then streamlit run app/main.py

After deployment, add your public Streamlit (or other) URL here by editing docs/index.html in the repository.
