CSV Analyst Assistant

A course project that turns plain-English questions about CSV data into a structured query, validates it against the dataset, and runs deterministic Pandas code. The language model is used for intent parsing only; execution stays in audited backend code.

View source on GitHub README & setup

What it does

  • Summary statistics (mean, max, count, …)
  • Grouped aggregations (e.g. by region, category)
  • Filtered aggregations (conditions on columns)
  • Time series at day, week, or month granularity

How it works

1. User question → LLM produces JSON matching a fixed schema 2. Validator checks columns, types, and task rules 3. Pandas runs the approved operation; charts use Matplotlib in the UI layer

This website (ahonakarim.github.io/ise547project) is the public project home: overview, links, and documentation. It is hosted on GitHub Pages, which serves static files only. The interactive query UI ships with the repository as a Python app (Streamlit) and runs on your machine or any host that can execute Python (see the README). That is a separate deployment from this site, not a replacement for it.

Run the interactive UI (optional)

For graders or demos who want the live Streamlit interface, clone the repo and run locally:

pip install -r requirements.txt then streamlit run app/main.py

You can also deploy that same app to Streamlit Community Cloud or Hugging Face Spaces; that is an optional companion to this site, not the primary project URL.
